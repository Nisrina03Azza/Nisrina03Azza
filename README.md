## 🗺️ Featured Projects

### [Space Utilization Suitability Evaluation — Yogyakarta City](https://github.com/Nisrina03Azza/space-utilization-suitability-evaluation)
Evaluated the suitability of existing space utilization per land parcel against the Detailed Spatial Plan (RDTR) of Yogyakarta City. Combined field survey data with multi-source spatial datasets, automated the analysis using QGIS Model Builder, and produced thematic maps following SNI cartographic standards at a 1:5,000 scale.

[![Suitability Evaluation Map](https://github.com/Nisrina03Azza/space-utilization-suitability-evaluation/raw/main/map/terban_suitability_eval.png)](https://github.com/Nisrina03Azza/space-utilization-suitability-evaluation)

### [Land Use Change Prediction Using Cellular Automata — Kabupaten Semarang](https://github.com/Nisrina03Azza/ca-LULC-semarang)
Improved a poorly converging CA-ANN model (Kappa 0.49 to 0.82) by diagnosing insufficient variable inputs and independently adding terrain layers. Modeled land use transitions between 2017 and 2019 and validated predictions against 2021 actual land cover.

[![Predicted Land Cover 2021](https://github.com/Nisrina03Azza/ca-LULC-semarang/raw/main/maps/LULC_2021_predicted_map.png)](https://github.com/Nisrina03Azza/ca-LULC-semarang)
